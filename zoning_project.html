The next 4 slides will show the data collection process for this project. In this part of the project we collected data from city zoning laws and zoning maps in order to build our zoning project dataset stored in multiple files for arcGIS.

Data Collection

In this first step of the zoning guide project the team and I are trying to build data in arcGIS like you see in the image above by using a cities zoning ordinance (as of sept. 2024 we've done 166 cities).

Starting with no data we first collect data from each cities zoning ordinance. We collect data on zoning district, bulk regulations, off-street parking, mobile homes, and other information regarding housing.

After collecting the data from the zoning ordinance we put all our data for each city on a single dataset.

In this final stage of the data collection phase we check to see if we have a shapefile for the given city we are working on(example above does). If we do then we merge the shapefile data with the encoded data in arcGIS otherwise we go to georeferencing which is our next step.

The next 4 slides will show the georeferencing process. We contacted the city hall members to see if they had a shapefile for there zoning map available. If they only had a pdf map then we had to convert that into a shapefile by georeferencing in arcGIS.

I start by converting the pdf zoning map in a tiff file usually done in photoshop. I open the city boundary file and outline each city boundary line in red. Lastly I import my new tiff file into the city boundary file and use the georeference function in arcGIS to align the tiff file image along the correct city boundary in this case West Union.

With the map layers aligned, I will proceed to create district polygons using ArcGIS's "Create Features" tool. By tracing new polygons over the district lines delineated in the TIFF file map, I will generate the necessary shapefile data for West Union's individual districts.

Shapefile Created

After maping each district I am usually left with about a 100 different polygons so I use pairwise dissolve or the merge tool to combine the polygons based on there correct district, lastly I check to make sure the shapes I created line up with the tiff map, I started with as demonstrated in the video here.
